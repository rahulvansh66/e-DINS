# e-DINS<sub>sub</sub>
Developing an evaluation metric "Effective degree of insensitivity w.r.t subjectivity (e-DINS<sub>sub</sub>)" for better ranking personalized text summarization models.

Paper accepted at **EMNLP'23**, a prestigious **ACL** conference (CORE'23 Rank **A\***)

[**Accuracy is not enough: Evaluating Personalization in Summarizers**](https://aclanthology.org/2023.findings-emnlp.169/)  
Rahul Vansh, Darsh Rank, Sourish Dasgupta, Tanmoy Chakraborty

**Guides:** Prof. Sourish Dasgupta (*DA-IICT*) and Prof. Tanmoy Chakraborty (*IIT Delhi*)
